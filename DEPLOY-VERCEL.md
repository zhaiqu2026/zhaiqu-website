# 🚀 Vercel 部署指南

## 部署步骤（5 分钟搞定）

### 第 1 步：访问 Vercel
打开：https://vercel.com

### 第 2 步：登录
点击 "Sign Up" 或 "Login"
**选择 "Continue with GitHub"**（用您的 GitHub 账号登录）

### 第 3 步：导入项目
1. 点击 "Add New Project"
2. 选择 "Import Git Repository"
3. 找到 `zhaiqu2026/zhaiqu-website`
4. 点击 "Import"

### 第 4 步：配置（自动识别）
Vercel 会自动识别配置：
- Framework Preset: Other
- Build Command: 不需要
- Output Directory: .

### 第 5 步：部署
点击 "Deploy"
等待 1-2 分钟，部署完成！

---

## 🌍 访问地址

部署成功后，您会获得：

1. **Vercel 域名**（全球可访问）
   ```
   https://zhaiqu-website.vercel.app
   ```

2. **自定义域名**（可选）
   - 在 Vercel 设置里绑定 `zhaiqu.fun`
   - 比 GitHub Pages 更快更稳定
